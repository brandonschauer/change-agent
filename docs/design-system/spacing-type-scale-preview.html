Spacing & Type Scale

A working preview of the proposed 4px spacing base and 8-step type scale, rendered in the Phase 0 web fonts — Source Sans 3 (text/UI) and Barlow Semi Condensed (display). Token names shown are the Style Dictionary CSS custom properties.

Type scale — 8 steps

A modular ramp from 12px to 40px on a 16px base, named font.size.100–.800. Steps 100–500 are the text/UI range (Source Sans 3); 600–800 are the display range (Barlow Semi Condensed). Toggle the preview font to judge each face across the scale.

Spacing scale — 4px base

Eleven steps, space.0–space.10, every value a multiple of 4px. Tight steps (4–24px) for component padding and gaps; larger jumps (32–64px) for section rhythm. Bars below are drawn at true size.
